# Menu-Driven-Python-Data-Analysis-Visualization-System
A Python-based menu-driven data analysis and visualization tool with user authentication, enabling CSV upload, data manipulation, statistical analysis, and automated chart generation using Pandas and Matplotlib.

# Features
> User Authentication
  -> Username/password login
  -> 3 login attempts
  -> Secure program access

> Data Upload
  -> Upload CSV files
  -> Sets the first column as index
  -> Error handling for invalid files

> Data Analysis Tools
  -> View Top 5 / Bottom 5 rows
  -> Extract data using loc or iloc
  -> View data type info
  -> Summary statistics (describe())
  -> Display column names
  -> Sort data (ascending/descending)

> Data Manipulation
  -> Delete rows (temporary/permanent)
  -> Delete columns (temporary/permanent)

> Data Visualization
  -> Bar Graph (Vertical/Horizontal)
  -> Line Chart
  -> Histogram
  -> Automatic NaN handling
  -> Custom axis labels & titles

> Data Export
  -> Export manipulated dataset to output.csv

> Tech Stack
  -> Python
  -> Pandas
  -> Matplotlib
